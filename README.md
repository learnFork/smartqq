# smartqq

一个基于smartQQ的QQ消息管理平台。fork项目，即可运行。

测试地址：[jcker.org](http://www.jcker.org)

##主要使用技术
###spring mvc 
    Jsp与controller数据交互
###spring data jpa 
    jparepository做数据库的crud操作
###hibernate 
    目前只是用来做ddl操作
###spring websocket 
    动态发送实时数据到页面
###h2数据库 
    方便部署
